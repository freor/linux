Create and open a shell script file named `my-script.sh`.
```sh
vim my-script.sh
```

Write down sciprts in the VIM editor and save the file.
`my-script.sh`
```sh
#!/bin/sh

<script1>
<script2>
<script3>
```

Change the permission and run the shell script file.
```
chmod 755 my-script.sh
./my-script.sh
```
